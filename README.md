# **2D游戏引擎——js**
* 基于**canvas**与**原生js**实现，逐步抽象封装**游戏引擎**
* 提供**资源管理，事件机制，场景抽象，精灵动画，粒⼦效果** 
* 实践**Code Gen**，**游戏参数自动配置**，动态调整游戏效果。
* 利用该引擎完成了 **Flappy Bird、飞机大战**等游戏
